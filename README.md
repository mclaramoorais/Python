#  Exercícios de Python 

Este repositório reúne implementações e resoluções de exercícios desenvolvidos em **Python**, com foco nos principais conceitos de **Estruturas de Dados e Algoritmos** 

## 📂 Estrutura do Projeto

| Arquivo                | Descrição                                                                                                                |
| ---------------------- |  ------------------------------------------------------------------------------------------------------------------------|
| `Q1_stack.py`          | Implementação das operações básicas de uma pilha seguindo o princípio LIFO (*Last In, First Out*).                       |
| `Q2_circular_queue.py` | Implementação de uma fila que reutiliza posições disponíveis na memória, seguindo o modelo FIFO (*First In, First Out*). |
| `Q3_priority_queue.py` | Estrutura em que os elementos são processados de acordo com sua prioridade.                                              |
| `Q4_linked_list.py`    | Implementação de listas utilizando nós conectados dinamicamente.                                                         |
| `Q5_sort_search.py`    | Implementação de algoritmos de ordenação e busca, como Bubble Sort e Busca Binária.                                      |
| `Q6_matriz_notas.py`   | Manipulação de estruturas bidimensionais para armazenamento e processamento de dados.                                    |
| `Q7_fatorial.py`       | Aplicação de funções recursivas, incluindo o cálculo de fatorial.                                                        |
